# 危険物乙4Pass プライバシーポリシー

最終更新日: 2026年6月19日

危険物乙4Pass（以下「本アプリ」）は、危険物取扱者試験 乙種第4類の学習機能、広告表示、アプリ内課金、利用状況の把握のために、端末上の学習データおよび第三者サービスが提供する識別子等を利用する場合があります。

本アプリは、ログインやアカウント作成を必要としません。

## 収集する情報

- 学習履歴、回答履歴、正誤履歴、苦手問題、連続学習日数など、アプリ内で保存される学習データ
- 文字サイズなどのアプリ設定
- 広告配信や広告効果測定に必要な識別子、広告表示に関する情報
- アプリ内課金の購入状態を管理するために必要な情報
- アプリの起動、学習開始、回答、模試完了、課金画面表示などの利用状況イベント
- クラッシュ、不具合、品質改善に必要な技術情報

本アプリは、氏名、住所、電話番号、メールアドレス、連絡先、正確な位置情報を、アカウント情報として独自に取得しません。

## 利用目的

- 問題演習、復習、学習進捗表示の提供
- 誤答優先学習、苦手問題、連続学習日数などの学習機能の提供
- 広告表示およびリワード広告による問題追加解放
- アプリ内課金の購入状態の確認、復元、管理
- アプリの利用状況分析、品質改善、機能改善
- 不具合調査

## 学習データの保存

学習履歴、回答履歴、苦手問題、連続学習日数などの学習データは、原則として利用者の端末内に保存されます。

本アプリは、学習履歴を管理するための独自アカウントサーバーを運営していません。端末の削除、アプリの削除、端末変更などにより、端末内の学習データが失われる場合があります。

## Firebase Analytics の利用

本アプリでは、アプリの利用状況を把握し、学習体験や機能品質を改善するために、Google LLC が提供する Firebase Analytics を利用する場合があります。

Firebase Analytics では、アプリの起動、学習モードの利用、分野ID、正誤、模試結果のスコア帯、課金画面の表示元など、アプリの利用状況に関するイベントを送信することがあります。

Firebase Analytics には、問題本文、選択肢本文、解説本文、自由入力、記述答案、氏名、メールアドレス、個別相談内容、Apple の決済識別子の生値、広告IDを送信しません。

Firebase Analytics により収集される情報は、Google のプライバシーポリシーおよび Firebase の規約に従って取り扱われます。

- Google プライバシーポリシー: https://policies.google.com/privacy
- Firebase におけるプライバシーとセキュリティ: https://firebase.google.com/support/privacy

## 広告について

本アプリでは、Google Mobile Ads を利用して、バナー広告、インタースティシャル広告、リワード広告を表示する場合があります。

リワード広告は任意で視聴する広告であり、視聴により追加の問題を利用できる場合があります。

Google Mobile Ads では、広告配信、広告効果測定、不正防止などのために、端末情報、広告識別子、利用状況に関する情報が利用される場合があります。これらの情報の取り扱いは、Google のポリシーに従って管理されます。

## アプリ内課金について

本アプリでは、非消耗型のアプリ内課金により、全問題の利用、広告非表示、詳細解説、選択肢ごとの解説、苦手問題モードなどの機能を提供する場合があります。

購入処理は Apple の仕組みを通じて行われ、購入状態の管理には RevenueCat を利用します。RevenueCat では、購入状態の確認、復元、課金機能の提供に必要な識別子や購入情報が取り扱われる場合があります。

## 第三者サービス

本アプリでは、広告配信、課金管理、利用状況分析、品質改善のために、以下の第三者サービスを利用する場合があります。各サービスで取り扱われる情報は、それぞれのプライバシーポリシーに従って管理されます。

- Firebase Analytics
- Google Mobile Ads
- RevenueCat
- Apple が提供する App Store、アプリ内課金、クラッシュログ、診断関連サービス

## 外部送信

広告配信、課金管理、利用状況分析、品質改善のため、第三者サービスに端末情報、広告識別子、購入情報、利用状況イベント、クラッシュ情報、診断情報などが送信される場合があります。

本アプリは、利用者が入力した氏名、住所、電話番号、メールアドレス、連絡先、正確な位置情報を独自に収集して外部送信する機能を持ちません。
